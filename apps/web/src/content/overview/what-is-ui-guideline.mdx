# What is UI Guideline?

UI Guideline is a curated reference of real-world design systems and their applied patterns.

## Why it exists

We collect high-quality implementations to showcase best practices and tangible patterns.

## How to use it

- Browse components to compare their anatomy and props.
- Explore overview docs like this page to understand philosophy and goals.
- Integrate design layers and code anatomy into your own system.
