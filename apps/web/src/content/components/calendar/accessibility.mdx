## Accesibilidad

- Usa roles `grid`, `row`, `gridcell` para la estructura del calendario.
- Los días deben anunciarse con `aria-label` (ej: "15 de septiembre 2025").
- Navegación con teclado:
  - Flechas ← ↑ → ↓ para moverse entre días.
  - PageUp/PageDown para cambiar de mes.
  - Shift + PageUp/PageDown para cambiar de año.
- Siempre resaltar el día enfocado con `:focus-visible`.
